# Persona 5 風 カットインジェネレーター（個人開発）

自分が好きなゲーム『ペルソナ5』のスタイリッシュなUI・エフェクトを再現したくて、Flutterを用いた個人開発に挑戦したプロジェクトです。
ユーザーが選んだ写真をペルソナ風のカットインにするアプリを目指していました。
https://youtu.be/ZTncuYKbLxE
この動画のようにペルソナ５のカットインはPCではつくれていたもののスマホアプリでは作れなかったためスマホアプリ版を開発しようと思って作成し始めた。

「透過WebM素材 ＋ ユーザー写真 ＋ 背景色」を
重ね合わせて1つの動画として保存する機能を実装する際、
技術的な問題から実装完了に至らなかった原因の分析記録です。

ios,android両方でリリースしようとしたためflutterを使用

結果として失敗

失敗理由
1 クロマキー透過がうまくいかず
ペルソナのカットインは一番前にカットインの枠　次にキャラクターの顔　次に真っ赤な背景というものです。
枠外と枠内がともに赤の背景のため動画などでクロマキー透過をおこなってしまうと背景が透明で残念なカットインになってしまう。
ffmpegやプログラムなどで透過処理を行おうとしたがうまくいかず…

2　動画と画像を重ねる機能が標準にない
flutterの画面上でおおよそこんな風になるという表示はできていたが、いざ動画と画像を重ね合わせようとするとうまくいかなかった。
androidやiosの標準機能はレイヤー順を正確に保って合成する処理が苦手なことをこの時になって知った。



## 🛠️ 技術スタック
- フロントエンド: Flutter (Dart)
- バックエンド / 画像処理: Python
- 外部ツール: ComfyUI (API連携検証)
